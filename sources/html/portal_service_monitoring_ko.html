Monitoring Dashboard

Total Server

관리중인 서버의 개수와 알람 발생 현황을 인지도가 높은 색감으로 가시성 있게 표현하였습니다.

Storage

서버의 스토리지(디스크) 용량과 현재 사용량을 인지하고 관리하도록 합니다.

Today’s Alarm Info

오늘 발생한 알람의 서버 및 관련정보를 표시하여, 바로 확인가능하도록 합니다.

Usage Top 5 (Line Chart)

CPU, Memory, Disk, Queue, Throughout 별로 사용량 상위 Top5 서버의 변동 추이를 점검할 수 있으며, 사용자가 원하는 정렬 기준과 수집 기간으로 손쉽게 설정하여 확인 할 수 있습니다.

Service Summary

관리 대상 서비스의 전체 현황에 대한 요약 정보 및 수치를 차트와 그래프로 표시하여 인지가 용이하도록 하여, 모니터링 담당자에게 필요한 정보를 효율적으로 제공합니다.

Top 5 List

5가지 주요 지표 중 선택한 지표 기준으로 상위 5개의 서버의 성능 현황을 모니터링할 수 있습니다.

Monitoring Dashboard

Alarm

선택한 서비스의 알람 발생 현황을 일자별, 기간별로 조회할 수 있으며, 알람의 발생과 종료 등의 진행상황과 알람 원인 분석을 위하여 알람 발생 시간 기준으로 메트릭 정보를 제공합니다.

Alarm / Custom Alarm

알람 목록 및 사용자 알람 목록을 표시하며 알람의 진행상태와 현황을 일자별/기간별로 선택하여 손쉽게 조회할 수 있으며, Open 되어 있는 알람 및 커스텀 알람은 Background를 각 컬러로 표시하고 있습니다.

Metric

서버에서 발생하는 메트릭 알람을 표시하여, 서버 상태를 점검할 수 있습니다.

CPU / Memory

서버의 CPU 및 Memory 사용량을 시간대별로 확인하여, 사용량이 많은 시간대와 저조한 시간대를 확인하고, 스케일을 조정하기 위해 필요한 정보를 제공합니다.
